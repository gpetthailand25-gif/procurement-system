ยังไม่ได้ตั้งค่า Firebase

ไฟล์นี้ต้องเชื่อมต่อกับฐานข้อมูล Firestore ของคุณเองก่อนใช้งาน

  1. ไปที่ console.firebase.google.com สร้างโปรเจกต์ใหม่
  2. เปิด Firestore Database (เลือก Start in test mode ไปก่อน)
  3. Project settings → Your apps → กดไอคอน </> เพื่อสร้าง Web app
  4. คัดลอกก้อนโค้ด firebaseConfig ที่ได้
  5. เปิดไฟล์นี้ด้วยโปรแกรมแก้ไขข้อความ ค้นหาคำว่า firebaseConfig แล้ววางค่าของคุณแทนที่
